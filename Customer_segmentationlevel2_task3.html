#Elbow Method Evaluation
inertia = []

for k in range(1, 12):
    kmeans = KMeans(n_clusters=k, random_state=45)
    kmeans.fit(churn_scaled)
    inertia.append(kmeans.inertia_)

plt.plot(range(1, 12), inertia, marker='o')
plt.title("Elbow Method")
plt.xlabel("Number of Clusters")
plt.ylabel("Inertia")
plt.show()

Explanation¶

The elbow method was used to determine the optimal number of clusters.

Inertia measures how tightly data points are grouped within clusters. As the number of clusters increases, inertia decreases.

The optimal number of clusters is identified at the "elbow point," where the rate of decrease slows significantly.

#Silhouette Score Evaluation
The silhouette score evaluates clustering quality.
This metric measures how similar data points are within their cluster compared to other clusters. Values that are closer to 1 indicate a well-separated clusters.
The optimal number of clusters corresponds to the highest silhouette score.

summary¶

The clustering analysis successfully segmented customers into distinct groups using K-Means. The optimal number of clusters was determined using the elbow method and silhouette score, ensuring reliable grouping. Cluster profiling provided deeper insights into the characteristics of each segment, while PCA and t-SNE visualizations confirmed clear separation between clusters. These findings demonstrate the effectiveness of unsupervised learning in uncovering hidden patterns within data. The results can be leveraged to enhance business strategies, particularly in customer segmentation, targeted marketing, and retention planning. Project Workflow & Key Steps

Optimal Cluster Selection:¶

To determine the ideal number of clusters, I applied:

  • The Elbow Method, which revealed a clear bend indicating the optimal cluster range

  • The Silhouette Score, which confirmed strong cluster separation and cohesion

These methods ensured that the clustering solution was both accurate and reliable.

• K-Means Clustering Implementation: Using the optimal number of clusters, the K-Means algorithm was applied to segment the dataset. Each data point was assigned to a cluster based on similarity, forming distinct customer groups.

Visualization & Insights from clustering¶

• PCA Visualization: Dimensionality reduction using Principal Component Analysis (PCA) enabled a clear 2D visualization of clusters, showing well-separated groupings. • t-SNE Visualization: To further validate clustering performance, t-SNE was used. This provided an even clearer separation of clusters, confirming that meaningful patterns exist within the data.

Cluster Profiling & Interpretation¶

Cluster profiling revealed distinct behavioral patterns across segments: One cluster represented high-value or highly engaged customers Another reflected moderate or average users A third indicated low-engagement or potentially at-risk customers

These insights are highly valuable for:

  • Targeted marketing strategies *Customer retention planning *Personalized service delivery
